Okay, so I believe we have a way forward for this issue that is the proper issue.
Issue:
Some epic games require a browser-based approval which is invoked on the game starting up.
Due to the limitations of the default wine browser, we are seeing issues where the default wine browser is unable to render the approval button and webpage correctly.
Solution:
We should install at first launch: Gecko, palemoon (browser)
Set the correct registry values so that palemoon is the default browser and has the associated libs in order to work
Note: we can always install another browser, but we know this one works.
We have confirmed that this will fix the issue.
